# CAT Examples

Mit meheren Datein

`cat file1 file2`

Line Number anzeigen

`cat -n filename`

Leere zeilen unterdrücken

`cat -s geeks.txt`

Ausgabe von _file1_ an datei _file2_ unten anhängen.

`cat file1 >> file2`

Ungekerte ausgabe

`tac filename`
